## 🌱 灵根相关

<details>
<summary>展开</summary>

<details>
<summary>灵根说明 📖</summary>

| 项目 | 内容 | 备注 |
| --- | --- | --- |
| 梯度 | 天灵根 → 异灵根 → 真灵根 → 五行伪灵根 → 伪灵根 → 废灵根 | 基础战力加成整体递减 |
| 作用 | 入宗限制、斗法加成、五行相克 | 境界、法宝、资源仍是核心 |
| 天灵根 | 金 / 木 / 水 / 火 / 土单属性 | 基础战力最高，但部分宗门受限 |
| 异灵根 | 风 / 雷 / 冰 | 克制关系更特殊 |
| 真灵根 | 通常为双灵根 | 限制较少，综合体验稳定 |
| 伪灵根 | 1至5个基础五行组合 | 五行齐全时有特殊加成 |
| 废灵根 | 无明显属性优势 | 前期弱，但并非不能玩 |
| 特殊灵根 | 血 / 毒 / 时间 / 空间 / 虚无等 | 多为 AI 演绎，不作为正式依据 |

</details>

<details>
<summary>战力加成 📈</summary>

| 灵根 | 加成 | 备注 |
| --- | --- | --- |
| 天灵根 | **+15%** | 当前最高基础加成 |
| 异灵根 | **+12% 至 +13%** | 风 / 雷 / 冰 |
| 真灵根 | **+10%** | 双灵根常见 |
| 五行伪灵根 | **+11%** | 金木水火土齐全时触发 |
| 伪灵根 | **+5%** | 普通伪灵根 |
| 废灵根 | 0 | 无加成 |

</details>

<details>
<summary>五行相克 ⚔️</summary>

| 克制 | 说明 | 备注 |
| --- | --- | --- |
| 金克木 | 基础五行克制 | 仅天灵根 / 异灵根可触发额外克制伤害 |
| 木克土 | 基础五行克制 | 额外伤害 **15%** |
| 土克水 | 基础五行克制 | 同上 |
| 水克火 | 基础五行克制 | 同上 |
| 火克金 | 基础五行克制 | 同上 |
| 冰克火 | 特殊克制 | 冰灵根压制火灵根 |
| 雷克五行 | 特殊克制 | 雷灵根克制金木水火土 |

境界压制仍是根本。修为差距过大时，五行克制难以逆转胜负。

</details>
